# 创建组件库

开发自己的组件库，并且上传到npm

## 初始化基础目录框架

首先创建我们的文件夹,和`npm`创建一样，使用`pnpm init`即可，当我们创建完成之后，会得到一个`package.json`

```json
{
  "name": "@seekhoo/ikun-ui",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "Seekhoo <hu@seekhoo.cn>",
  "license": "ISC"
}

```
