# 什么是 Nuxt

- 支持 Vue
- 采用 h3 实现部署可移植性
- 支持 CSR SSR SSG

## 发展

- 2016 年 10 月 25 创建 ，基于 Vue2 webpack2 node 和 express
- 2018.1.9 发布 Nuxt.js 1.0
- 2018.9.21 发布 Nuxt.js 2.0
- 2022.12.16 Nuxt.js 3.0

## 特点

- 使用 Vue3+Vite
- 自动导包
- 基于规范目录结构，自动导入组件插件
- 约定式路由 目录结构即路由
- 支持多重渲染模式
- 利于搜索引擎优化
