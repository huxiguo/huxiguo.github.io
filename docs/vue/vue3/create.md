# 创建项目

## 1. 使用 vue-cli

[**vue-cli 官方文档**](https://cli.vuejs.org/zh/)

```bash
## 创建vue3的vue-cli版本必须大于4.5.0

## 创建

vue create vue3_test

## 启动
cd vue3_test
npm run serve
```

## 2. vite

[**vite 官方文档**](https://cn.vitejs.dev/)

```bash
## 创建工程
npm init vite-app <project-name>

## 进入工程
cd <project-name>
## 安装依赖
npm i
## 运行
npm run dev
```
